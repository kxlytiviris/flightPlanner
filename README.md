# flightPlanner
Mi primer projecto en GitHub

Flight PLanner es un planificador de vuelos desarrollado en C++.
Permite al usuario a seleccionar un destino, indicar maletas extra y obtener un costo total estimado del vuelo.

Features:
Lista de destinos y precios base, Cálculo automático del precio, Array de destinos y array de precio, Uso de constantes, Menú con estructura switch, Funciones separadas para mejor organización, Uso de librerías adicionales (<cmath>, <cstdlib>) , Recomendaciones simples basadas en el precio


Como utilizar?
Ejecuta el programa, Selecciona una opción del menú, Ver destinos, Elegir un destino, Indicar maletas, Calcular costo total, El programa mostrará el costo final del vuelo, Sal del programa cuando termines.
